# Steps-Market
O mercadinho “Bom Preço” começou a ficar muito cheio, então ao perceber tal erro, criou um sistema para controlar a entrada e saída de clientes. Na entrada, cada pessoa recebia um cartão, e havia um limite de clientes dentro da loja. Quando atingia esse limite, os demais aguardavam do lado de fora. Na saída, os cartões foram devolvidos, ajudando no controle.
